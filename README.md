OpenForest4D API Workbench

This repository provides practical examples for using the OpenForest4D API, along with code samples that can be used directly or adapted into your own workflows.

### Examples

[OF4D_Meta_WRI_CHM_API_example.ipynb](https://github.com/OpenForest4D/API_workbench/blob/main/OF4D_Meta_WRI_CHM_API_example.ipynb) - This notebook walks through querying the OpenForest4D API to retrieve and visualize a clipped Meta/WRI Version 2 Canopy Height Model (CHM) for a user-defined area of interest (AOI). Data are downloaded as a GeoTIFF in EPSG:3857 (Web Mercator), then visualized, with a canopy height histogram generated for exploratory analysis.
